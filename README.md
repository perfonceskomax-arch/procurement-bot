# Бот закупок для Макса

Читает заявки из группы, распознаёт что нужно купить, ищет цены и скидывает результат в личку.

## Деплой на Railway

### 1. Залить код на GitHub
- Создай новый репозиторий на github.com
- Загрузи файлы: bot.py, requirements.txt, railway.toml

### 2. Настроить Railway
- Зайди на railway.app
- New Project → Deploy from GitHub repo → выбери репозиторий
- Перейди в Variables и добавь:

```
BOT_TOKEN=твой_токен_от_BotFather
MY_TELEGRAM_ID=736275799
ANTHROPIC_API_KEY=твой_ключ_от_anthropic
```

### 3. Получить Anthropic API ключ
- Зайди на console.anthropic.com
- Зарегистрируйся
- API Keys → Create Key
- Скопируй и вставь в Railway

### 4. Добавить бота в группу
- Открой группу "Предзаказы масмаркет Согласования"
- Добавить участника → найди своего бота по username
- Дай боту права читать сообщения (обычный участник)

## Как работает

1. Кто-то пишет в группу "нужны тарелки 50 штук"
2. Бот через ИИ понимает что это заявка на закупку
3. Ищет цены в интернете
4. Скидывает тебе в личку структурированный результат с ссылками

Если кидают ссылку — бот пересылает её тебе с пометкой что нужно проверить цену.
